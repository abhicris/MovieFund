# Alpha Testnet Version Notice

## Current Status

This is an **Alpha Testnet Version** of MovieFund. The platform is currently in development with core features being built and tested.

- Real-time updates and notifications

- KYC/AML compliance

### 3. Movie Company Profiles
- Production company profiles
- Company verification and credentials
- Portfolio of funded movies
- Financial track record
- Contact and communication tools

### 4. Investor Profiles
- Investor dashboard
- Portfolio tracking
- Investment history
- Returns tracking and statements
- Tax documentation
- Communication preferences

### 5. Onchain Records (Blockchain Ledger)
- **Important**: Blockchain is used as a **ledger** for record-keeping, not for transactions
- Immutable records of investments
- Transparent ownership records
- Returns distribution records
- Audit trail for all transactions
- Smart contracts for automated record-keeping

## Technology Stack (Planned)

- **Backend**: Node.js/Express or Next.js API Routes
- **Database**: PostgreSQL for transactional data
- **Blockchain**: Ethereum/Polygon for ledger records
- **Authentication**: NextAuth.js or Auth0
- **Payment Processing**: Stripe/Razorpay integration
- **File Storage**: AWS S3 or Cloudinary

## Current Limitations

- No user authentication (demo mode only)
- Mock data for movies and investments
- No actual payment processing
- No blockchain integration yet
- No API endpoints
- No user profiles or dashboards

## Important Notes

1. **Blockchain Usage**: The blockchain will be used exclusively as a ledger for maintaining immutable records. All financial transactions will be processed through traditional payment gateways.

2. **Data Privacy**: User data and financial information will be stored securely in traditional databases, with only investment records and ownership data stored on-chain.

3. **Compliance**: The platform will comply with all relevant financial regulations and securities laws in jurisdictions where it operates.

4. **Testing**: This alpha version is for testing and demonstration purposes only. No real investments can be made at this stage.
