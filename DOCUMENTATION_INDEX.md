## How to Update Investor Deck Link

When the Gamma presentation is ready:

1. Update the link in `/app/docs/page.tsx`:
   ```tsx
   link: "https://gamma.app/your-presentation-link"
   ```

2. Remove `comingSoon: true` flag

3. The link will automatically appear on the docs page
